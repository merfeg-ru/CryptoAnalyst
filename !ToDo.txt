Бизнесмен
1. Запрашивает у анализатора данные может че купить через Api
2. Говорит торговцу что купить через Api торговца
3. Говорит торговцу когда продать через Api торговца
4. Помнит о выставленных ордерах -> работа с БД
5. Следит за курсом
	-> текущие цены из шины
6. Решает когда продать, пытается получить заданный профит
7. Api статистики модуля
	- сколько открыто ордеров за сутки
	- прибыль за сутки
	- прибыль за т.месяц
	- прибыль за т.год
